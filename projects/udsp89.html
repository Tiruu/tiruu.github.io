Project description

This project is the result of my third-year university internship. I was asked to create an innovative, fun and interactive way of talking about the UDSP 89 association.

My task was as follows : "Development of an interactive device in the form of a mobile application and website with the aim of communicating with the association's members."

I chose to create a Point & Click game that would be easy to understand yet fun to play.

I modeled all the scenery, as well as all the game's props. I also learned how to create, rig and animate a character.

The game includes 7 scenes, all containing information about the association's operations

I also developed a website to host the game as well as the in-game informations, so everyone can still get the informations.
